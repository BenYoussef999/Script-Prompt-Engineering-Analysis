# Script-Prompt-Engineering-Analysis

This code was made to make analysis on the accuracy of the AI agents nowadays.
It strats off by generating multiple random questions in different fields(Hitory,Maths,Physics,etc), then it transforms the questions to three different prompt echniques (zero-shot, few-shot, CoT) ,then it sends an api call to GPT-oss to ask the questions ans lastly it finishes by recording the answers in a excel file for further analysis and making dashboards
